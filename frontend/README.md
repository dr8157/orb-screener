# ORB Institutional Screener - Frontend

A premium React dashboard for real-time ORB signal tracking.

## Quick Setup

```bash
cd frontend
npm install
npm run dev
npm run build
```

Frontend: `http://localhost:5173`
Backend: `http://localhost:8000`

## Features

- 🟢 Live connection status
- 📊 Real-time rankings table  
- 📈 Sparkline charts
- ⚙️ Admin configuration
- 🌙 Dark premium theme
